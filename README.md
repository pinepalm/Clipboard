# Clipboard
 这是一个.NET使用[MiniBlink](https://github.com/weolar/miniblink49)内核进行界面渲染的剪贴板程序。
 使用的MiniBlinkNet接口项目地址:[https://gitee.com/kyozy/miniblinknet](https://gitee.com/kyozy/miniblinknet)

 html使用了Vue.js和Vant组件。
